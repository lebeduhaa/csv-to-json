# Csv to json parser

## Usage as a command line tool

git clone https://github.com/lebeduhaa/csv-to-json.git
cd cvs-to-json
npm link
csvToJson [options]

## Common usage

git clone https://github.com/lebeduhaa/csv-to-json.git
cd cvs-to-json
npm rum [options]

## Install as a global tool

npm install -g csv-to-json-lebeduha
csv-to-json-lebeduha [options]

### Options

--sourceFile
--resultFile
--separator
